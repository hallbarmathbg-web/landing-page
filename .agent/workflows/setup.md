# Workflow: Setup

Initialize the development environment.

## Steps

1. **Install Dependencies**:
   - For Node: `npm install`
   - For Python: `python -m venv venv` and `source venv/bin/activate` followed by `pip install -r requirements.txt`
2. **Setup Environment Variables**: Copy `.env.example` to `.env`.
3. **Verify Installation**: Run `npm test` or `pytest`.
